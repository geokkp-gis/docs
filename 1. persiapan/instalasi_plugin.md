# Instalasi Plugin GeoKKP-GIS

Aplikasi GeoKKP-GIS dibuat dalam bentuk plugin di QGIS untuk memanfaatkan kekayaan fungsi analisis dan dukungan data geospasial yang dimiliki oleh QGIS.
